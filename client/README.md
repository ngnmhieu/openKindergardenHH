#### Running the client
Was wir brauchen:
 - node.js ( [siehe nodejs.org](http://nodejs.org) )

Beim ersten mal:
 - npm install -g grunt grunt-cli bower
 - npm install
 - bower install

Client start:
 - grunt serve



### Tech stack
 - AngularJS
 - mapbox.js
 - [angular-leaflet-directive](http://tombatossals.github.io/angular-leaflet-directive/#!/examples/geojson)
